# Project Overview

This is the wiki home page — not a workflow step, just a plain markdown page.

## What this workflow does

This example demonstrates a simple 4-step agentic workflow:

1. **Research** — gather requirements
2. **Plan** — design the solution
3. **Implement** — write the code
4. **Review** — validate the output

Each step is a markdown file with YAML frontmatter that Flow-Claw reads to build the workflow view.
